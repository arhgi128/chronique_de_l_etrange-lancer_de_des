# Hong-Kong Les Chroniques de l'étrange - Lancer de dés

Ce projet est une **solution non officielle** de gestion des lancer de dés pour le jeu de rôle Hong-Kong Les Chroniques de l'étrange par Romain D’Huissier, aussi auteur des romans éponymes, et Cédric Lameire, publié par Antre Monde éditions. https://antre-monde.com/les-chroniques-de-letrengae/

## Présentation

Il se présente sous forme d'un one-page HTML pour être utilisé sur un ordinateur ou une tablette à votre table de jeu. Il n'a pas été prévu pour être intégré dans d'autres solutions comme des 

Il permet de 
- définir les joueuses et joueurs
- initialiser le loksyu et faire le rituel des personnages en cours de partie
- utiliser le loksyu pour un flash-back ou pour gagner du hei
- Sélectionner un joueur, faire un lancer: sélectionner, l'élément, le nombre de dés, faire le lancer puis gérer toutes les utilisations de Tin Ji, du Loksyu et des dés fastes et néfastes
- utiliser la Tin Ji pour retirer des dés du Loksyu ou pour les autres effets prévus en jeu

Les dés peuvent être lancés soit par le bouton "Lancer les dés" soit par clic/tap and drag.

## Librairies et remerciements

Ce logiciel repose sur la solution Teal Dice : https://emanchado.github.io/3d-die-roller/dice/ par Anton Natarov qui utilise les librairies suivantes
- Three.js
- Cannon.js

Il m'a été inspiré par l'adaptation de Teal Dice faite sur le site non officiel https://chroniques-etrange.fr/dice_chron_etrange/dice/
